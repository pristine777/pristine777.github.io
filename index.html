COMPARISON OF PID AND LQR BASED QUAD-COPTER CONTROLLER DESIGNS

The objective of this project was to study the functionality of the Linear-Quadratic-Regulator based controller design for highly non-linear systems. The major hurdle in the implementation was the interpretation of the non-linear dynamics of the Quad-copter system. It included understanding the theory behind 3-Dimensional flight dynamics. The performance of the LQR controller is compared with the conventional PID controller to record the difference in performance. This project is the replication of the results (for course-work purposes) of an already published IEEE paper which has been cited in the project report.

ANALYSIS OF INTELLIGENT PID CONTROLLER USING MULTI-LAYER NEURAL NETWORK

In the conventional PID controller, the controller gains are fixed for an operating point. In this project, a neural network is designed which determines the PID controller gains for every time step based on the current available output. This allows better control over the system output.

This project is the replication of the results (for course-work purposes) of an already published IEEE paper which has been cited in the project report.
NOTE: I was not able to simulate all the conditions. The controlled response for a non-linear system failed. I have plans to re-write the code to simulate the non-linear system.

In process for creating useful documentation for the project.

DISTURBANCE CONTROL OF FLUID CATALYTIC CRACKING UNIT (FCCU)

This project taken as part of the course-work involved simulating the process of an Fluid Catalytic Cracking Unit (FCCU) and designing control loops for the disturbance control due to changes in feedstock composition, feed preheat temperature and ambient temperature.

The process models were taken from an IEEE paper referenced in the project report. The project involved running the RGA / SVD analysis to determine the control loops, and designing a feedback control for maintaining stability of the system around the operating point. Simulation Softare used is VISSIM

REAL TIME SCHEDULER DESIGN & PROGRAMMING IN AN ARM ENVIRONMENT
This was a group project to design a clock driven scheduler to allow execution of periodic tasks (with known execution time) in fixed order in every frame. The processor execution frame size is calculated based on the available set of jobs / tasks to execute. The scheduler is designed such that, based on the available free time in each frame, the frame size is either increased to accommodate the task or task is scheduled in the next frame.

PATH FINDING ROBOT USING ROBO-VISION
The aim of this project is to develop a path finding robot which is capable of avoiding obstacles by itself using a USB web-cam interface and an AT89C2051 micro-controller. The images are captured from the camera in real time and are transferred to the computer for processing. MATLAB was used for image processing in order to determine obstacles using edge detection. Based on the image analysis, control commands were provided to the micro-controller on the robot through the RS-232 serial interface. This project was taken up as part of the Final year project during the Bachelor's Degree.
No electronic documentation available for this project.


8051 MICRO-CONTROLLER BASED FIRE MONTIORING & CONTROL SYSTEM
This project was the final year project for my Diploma in Digital Electronics. The project involved programming the micro-controller in assembly language. 8-channel thermocouple sensor array was used to measure temperature at 8 different locations. Once a fire is sensed by any channel of the thermocouple, the alarm for the respective channel was triggered and a control relay was activated.
No electronic documentation available for this project.
